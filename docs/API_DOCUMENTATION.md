# API Documentation

## Endpoints

### GET /api/students/
- Retrieves a list of all students.

### POST /api/students/
- Registers a new student.

### GET /api/students/{id}/
- Retrieves a student by ID.

### PUT /api/students/{id}/
- Updates a student by ID.

### DELETE /api/students/{id}/
- Deletes a student by ID.
